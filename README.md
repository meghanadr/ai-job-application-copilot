# AI Job Application Copilot

An AI-powered application that helps job seekers analyze resumes, compare them with job descriptions, identify skill gaps, generate personalized learning roadmaps, create interview questions, improve resumes, and draft cover letters, emails, and LinkedIn messages.

## Features

- Resume Analysis
- Job Description Analysis
- ATS Match Score
- Resume Improvement
- Learning Roadmap
- Interview Questions
- Cover Letter Generator
- Job Application Email Generator
- LinkedIn Message Generator

## Tech Stack

- Python
- Streamlit
- LangChain
- OpenAI GPT-4o Mini
- PyPDF
- python-dotenv
